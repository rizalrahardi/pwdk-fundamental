problem :
mencari luas persegi panjang

hint :
mengukur atau mengetahui panjang tiap sisi persegi panjang
ambil sisi panjang dan lebar persegi panjang
jika diketahui panjang sisi 20 dan lebar sisi 10 maka sesuai rumus luas persegi panjang
luas = p x l
definisikan dalam variabel length = 20
width = 10

solutions in pseuodocode :
olah data variabel yang sudah didapatkan dengan formula menghitung luas persegi panjang
const luasPersegiPanjang = length * width ;

problem :
menghitung keliling persegi panjang

hint :
mengukur atau mengetahui panjang tiap sisinya
ambil sisi panjang dan lebarnya kemudian diolah dengan formula/rumus
keliling persegi panjang = 2 x (p+l)
definisikan dengan variable
const keliling persegiPanjang = 2 * (length + width)

solutions in pseuodocode :
olah data variable dengan formula menghitung luas persegi panjang
const luasPersegiPanjang = 2 * (length + width)
